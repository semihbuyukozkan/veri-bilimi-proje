# E-Ticaret Müşteri Davranışı ve Segmentasyon Analizi

Bu proje, bir e-ticaret platformunun satış verilerini analiz ederek müşteri davranışlarını anlamlandırmayı ve makine öğrenmesi yöntemiyle müşteri segmentasyonu gerçekleştirmeyi amaçlayan bir veri bilimi dönem projesidir. Proje sürecinde yapay zeka kodlama asistanlarından "vibe coding" yaklaşımı ile birincil geliştirme ortağı olarak yararlanılmıştır.

## 🚀 Projeyi Çalıştırma Talimatları
1. Proje deposunu yerel bilgisayarınıza indirin veya Google Colab üzerinde açın.
2. Gerekli kütüphanelerin yüklü olduğundan emin olun (`pip install kagglehub pandas scikit-learn matplotlib seaborn`).
3. `1306230099.ipynb` dosyasını sırasıyla tüm hücreleri çalıştırarak (Run All) yürütebilirsiniz. `kagglehub` kütüphanesi veri setini Kaggle üzerinden otomatik olarak indirecektir, harici bir veri yüklemesine gerek yoktur.
